Hello! This is a TypeScript rewrite of the original project.

Current layout:
- `front/` - Vue 3 + Quasar frontend
- `back/` - TypeScript backend with Express + WebSocket

Run locally:
- backend: `cd back && npm install && npm run dev`
- frontend: `cd front && yarn install && yarn dev`

The backend is currently a TS in-memory implementation that preserves the main API contracts used by the frontend.
